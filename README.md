# multi-lang-jenkins-demo

Monorepo pour formation Jenkins contenant 5 mini-projets :
- python-app (Flask + pytest)
- springboot-app (Spring Boot + Maven + JUnit)
- node-app (Node.js + Jest)
- php-app (PHP + PHPUnit)
- go-app (Go + go test)

Chaque projet contient :
- code minimal
- test unitaire simple
- Jenkinsfile (pipeline `agent any`)
